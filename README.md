# Reign
Reign is a website that offers real-time chat, moderation roles, an emoji reaction system, and a trading system.  (One of my first big HTML/CSS/JS Projects, Prob went too far ahead :/)

Local Based PFP so you can only see it, met for the needs of a user with Spark Firebase plan

Design
Contains a smooth, simplistic "ERROR" or 404 page, simulating a Chrome error page, where if you click "Connection Timed Out" 5 times, it redirects you to a galaxy-themed page, with titles that have glowy effects following the colors blue and purple. As you press the buttons to get in, they highlight and smoothly transition
to the desired destination. It uses Firebase Authentication for secure login and registration, with real-time form validation and feedback.

The Message Page consists of a chat server with real-time chat, direct messages, and trading. Trading history is saved regardless of a no-deal or deal. The style of the Message Page stays consistent with the login page and a moderation system includes admin where you can easily delete and purge messages, with the admin's input of how many messages purged, suspending, banning, deleting accounts, and muting accounts come along with admin permission. Text channels contain custom names and extra information on the side about them. You can also configure
Your status to be away if you're online and offline status is automatically set. Pinging is highlighted and shows up as a notification if you're pinged from a different tab. Also in the settings there is local based profile configuration where you can choose any image below 2MB
and select it to show for you exclusively. (Note that Spark has no Storage on Firebase, hence why)
